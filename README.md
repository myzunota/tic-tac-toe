# GRID DUEL — ○× ゲーム

シングルファイルの HTML で動作する ○× ゲームです。  
クラシックな **2D モード（3×3）** と、立体的な **3D モード（3×3×3）** の 2 種類を収録しています。  
インストール不要。ブラウザで HTML ファイルを開くだけで即プレイできます。

## 特徴

- **モード選択ホーム画面** — 2D / 3D を選んでスタート。3D はプレイヤー数（2〜3 人）も選択可能
- **スコア記録** — ゲームをまたいで勝利数を保持（リセットも可能）
- **キーボード操作対応** — 数字キー `1`〜`9` でマウスなしプレイ
- **3D キューブ回転** — ドラッグまたはボタンで視点を自由に変更
- **勝利ハイライト** — 揃った 3 マスをアニメーション付きでハイライト
- **依存ライブラリなし** — 外部フレームワーク・サーバー不要

## ゲームルール

### 2D モード — クラシック 3×3

標準的な ○× ゲームです。

- プレイヤーは **○（先攻）** と **✕（後攻）** の 2 人
- 3×3 のマス目に交互に記号を置く
- **縦・横・斜めのいずれかで 3 マスを一直線に揃えた側の勝利**
- 全マスが埋まっても揃わなければ **引き分け**

```
 ○ │ ✕ │ ○
───┼───┼───
 ✕ │ ○ │ ✕    ← ○ の勝利（右斜め）
───┼───┼───
 ✕ │   │ ○
```

---

### 3D モード — 立体 3×3×3

3 層（Bottom / Mid / Top）からなる立体盤面を使った拡張版です。

- プレイヤーは **2 人または 3 人**（開始前に選択）
  - 2 人: ○（P1）vs ✕（P2）
  - 3 人: ○（P1）、✕（P2）、△（P3）の順に手番が回る
- 駒は **上層（TOP）から投入**し、その列で空いている一番下のマスに落下する（重力落下方式）
- **縦・横・奥行き・斜め方向で 3 マスを一直線に揃えた側の勝利**

#### 勝利ラインの例

```
同一層の横一列:   (0,0,0)-(1,0,0)-(2,0,0)
同一列の縦一列:   (0,0,0)-(0,0,1)-(0,0,2)    ← 奥行き方向
層をまたぐ斜め:  (0,0,0)-(1,1,1)-(2,2,2)    ← 空間対角線
```

全 27 マスで **49 通りの勝利ライン** が存在します。

## 操作方法

### 共通操作

| 操作 | 内容 |
|---|---|
| **← ホーム** ボタン | ゲーム中断してホーム画面へ戻る |
| **新しいゲーム** ボタン | スコアを維持したまま盤面をリセット |
| **スコアをリセット** ボタン | スコアと盤面を両方リセット（確認ダイアログあり） |

---

### 2D モード操作

| 操作 | 内容 |
|---|---|
| マス目をクリック | 手番プレイヤーの記号を置く |
| キー `1`〜`9` | テンキー配置でマス目に入力（左上=`1`、右下=`9`） |
| **先攻を選択** ドロップダウン | 次のゲームから先攻プレイヤーを切り替え |

**キー配置（テンキー準拠）:**

```
7 │ 8 │ 9
──┼───┼──
4 │ 5 │ 6
──┼───┼──
1 │ 2 │ 3
```

---

### 3D モード操作

#### 駒の投入

| 操作 | 内容 |
|---|---|
| 上層入力グリッドのボタンをクリック | 対応する列に駒を落下投入 |
| キー `1`〜`9` | ボタンと同じ列に投入（左上=`1`、右下=`9`） |

> 列が満杯の場合はボタンが無効化され、投入できません。

#### キューブ視点操作

| 操作 | 内容 |
|---|---|
| キューブ上でドラッグ | 任意方向に自由回転 |
| `⟲` ボタン | Y 軸を左に 30° 回転 |
| `⟳` ボタン | Y 軸を右に 30° 回転 |
| `↺` ボタン | 初期視点にリセット |

## 画面構成

```
ホーム画面
├── モード選択カード（2D / 3D）
├── プレイヤー数選択（3D 選択時のみ表示）
└── START GAME ボタン
    │
    ├── 2D ゲーム画面
    │   ├── ← ホームボタン
    │   ├── 3×3 ゲーム盤
    │   ├── 手番・状態表示
    │   ├── スコアボード（○ / ✕）
    │   └── 操作ボタン群
    │
    └── 3D ゲーム画面
        ├── ← ホームボタン
        ├── 3D キューブビュー（ドラッグ回転）
        ├── 視点操作ボタン
        ├── 上層入力グリッド（9 ボタン）
        ├── 手番・状態表示
        ├── プレイヤーインジケーター
        ├── スコアボード（P1〜P3）
        └── 操作ボタン群
```

## 技術仕様

| 項目 | 内容 |
|---|---|
| 実装 | HTML5 / CSS3 / Vanilla JavaScript (ES2020+) |
| 外部依存 | Google Fonts（`Orbitron`, `Exo 2`, `Share Tech Mono`）※オフライン時はシステムフォントにフォールバック |
| 3D 描画 | CSS `transform-style: preserve-3d` による CSS 3D |
| 勝利判定（3D） | 全 49 ライン網羅（起動時に自動生成、グローバル汚染なし） |
| ファイルサイズ | 単一 HTML ファイル、約 30 KB |
